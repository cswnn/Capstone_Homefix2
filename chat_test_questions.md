### 4️⃣ 단계별 대화 시나리오 테스트 🧪

1. "가스레인지 기름때 제거법" (답변)

## 📊 테스트 체크리스트

### 구체성 판단 테스트

- [ ] 구체적인 질문은 바로 답변 제공
- [ ] 애매한 질문은 추가 질문 생성
- [ ] 추가 질문 후 구체적 답변 시 최종 답변 제공

### 문맥 인식 테스트

- [ ] 독립적인 새 질문은 새로 처리
- [ ] 이전 대화 참조 질문은 문맥 고려
- [ ] 대화 히스토리 유지 확인

### 준비물 링크 테스트

- [ ] 구체적인 질문의 경우 준비물 링크 제공
- [ ] 필수/선택 준비물 구분 확인
- [ ] 준비물 없음일 경우 링크 없음 확인

### 오타 처리 테스트

- [ ] 오타가 있어도 올바른 단어로 수정하여 답변
- [ ] 오타를 지적하지 않고 자연스럽게 처리

---

## 🎯 우선순위 테스트 순서

1. **1차** - 구체적인 질문 10개 테스트
2. **2차** - 애매한 질문 10개 테스트
3. **3차** - 문맥 기반 질문 5개 + 이전 대화
4. **4차** - 전체 시나리오 테스트
